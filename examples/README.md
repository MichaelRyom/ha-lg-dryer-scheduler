# Examples

Working examples you can copy into your Home Assistant instance.

| File | Where to paste |
|---|---|
| [`automation_cheapest_electricity.yaml`](automation_cheapest_electricity.yaml) | Settings → Automations → Create → Edit in YAML |
| [`dashboard.yaml`](dashboard.yaml) | New empty dashboard → Edit → Raw configuration editor |

In all examples, replace `torretumbler` with your dryer's entity prefix (whatever HA generated from your dryer's alias in the LG ThinQ app).
